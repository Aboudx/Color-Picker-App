## 📖 About

This project was created as a learning exercise to practice **TrackBar** controls and RGB color manipulation in **Windows Forms**.

---

## ✨ Features

- 🎨 Live color preview
- 🎚️ RGB sliders
- 🔢 HEX & RGB values
- 📋 Copy HEX color
- 🎲 Generate random colors
